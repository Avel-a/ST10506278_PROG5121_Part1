## Project Overview

This is chat application with a secure login system, the program checks if the user's login details are correct before granting access. This system consist of 3 classes which are(Login.java, MainApp.java and LoginTest.java), each of this class works differently from each. Login.java contains all the condition the user must follow. MainApp.java uses Scanner to prompt the user, read the user's input and display success or an error message. LoginTest.java uses Junit to test if Login.java works correctly.
